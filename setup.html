Setup Guide

Quick start steps to get Quant Manager running inside Quantower.

Install Configure Verify
1. Install the plugin
Build or copy the plugin output into your Quantower plugins directory.
2. Launch Quantower
Open Quantower and enable the plugin from the scripts/plugins list.
3. Configure accounts
Open the Edit Account panel to set limits, formulas, and scheduled flatten times.
4. Set templates
Create templates for consistent limits and formulas, then apply them in bulk.
5. Verify enforcement
Use a demo account to confirm locks, flattening, and daily resets work as expected.
Need help with installation? Contact support or check the main site.
